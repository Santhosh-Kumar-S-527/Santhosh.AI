# 🚀 Santhosh Kumar - AI-Powered Personal Portfolio

Welcome to my personal portfolio! This is a modern, highly interactive, and responsive web application built using **React (Vite)** to showcase my skills, projects, and achievements in Artificial Intelligence, Data Science, and Web Development.

## 🌟 Live Demo
👉 **[View Live on Vercel](https://your-portfolio-link.vercel.app)** *(Replace with your actual Vercel URL)*

## 🛠️ Core Features

- **Dynamic Homepage**: Features a custom typewriter effect hook, an automated education timeline, and precise progress bars representing technical skills (Python, AI/ML, Data Science).
- **Intelligent AI Chatbot Assistant**: A globally floating conversational interface responding to interactive queries regarding my projects, skills, and background using deterministic NLP-style keyword matching.
- **Global Light/Dark Theme Engine**: Equipped with a full-scale native CSS variable conversion system, letting users toggle between an immersive Dark Mode and a crisp Light Mode, completely stored via `localStorage`.
- **AI Resume Analyzer**: A flagship mock-AI utility allowing users to simulate a parsing engine on their resumes to generate actionable suggestions and AI scores.
- **Interactive Projects Dashboard**: Sort and filter projects instantly by categories (*AI, NLP, Data Science, MERN*) to easily view detailed pages of my work.
- **Robust Professional Profiles**: Dedicated layouts highlighting stats from LeetCode, HackerRank, CodeChef, and Skillrack, along with earned certifications in C++, Java, and Python.

## 💻 Tech Stack
- **Frontend Framework**: React 18 (Vite)
- **Routing**: React Router DOM (v6)
- **Styling**: Pure CSS (Glassmorphism, CSS Variables, Native Keyframes)
- **Iconography**: Lucide React
- **Deployment**: Vercel (configured with `vercel.json` for seamless SPA routing)

## 🚀 Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Santhosh-Kumar-S-527/portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running the App
Start the local development server:
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) in your browser to view the application.

## 📁 Project Structure

- `/src/components` - Reusable UI elements (Navbar, Footer, Chatbot, Resume Analyzer)
- `/src/pages` - Core top-level layout routes (Home, Projects, Coding Profiles, Contact)
- `/src/utils` - Centralized data models (e.g., `projects.js`, `analyzer.js`)
- `/src/assets` - Static media including custom project screenshots and PDFs.
